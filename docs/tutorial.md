# Team Setup

This document covers the required software installs for documentation and
Word-to-Markdown conversion.

---

## Install MkDocs and Required Plugins

MkDocs is used to build and publish the project documentation website.

```bash
pip install mkdocs
pip install mkdocs mkdocs-material mkdocs-awesome-pages-plugin mkdocs-mermaid2-plugin
pip install mkdocs-material
pip install mkdocs-awesome-pages-plugin
pip install mkdocs-mermaid2-plugin
```

## Steps to add md files

- Create a new file in format: YYYY-MM-DD.md in the /docs/progress folder
- Go to  index.md located in the /docs/progress folder
    - in this file, add your new file as such -> - [Week #] (YYYY-MM-DD.md)
- Push the changes once completed
